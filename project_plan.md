# ほぐし処らくっと LP

## 1. プロジェクト概要
福島県会津若松市にあるリラクゼーションサロン「ほぐし処らくっと」の1ページランディングページ。30〜40代女性をターゲットに、和モダンで落ち着いた雰囲気のデザインで、もみほぐし・ヘッドケア・リフレクソロジーのメニューを紹介し、予約獲得を目的とする。

## 2. ページ構成
- `/` - LP（全セクションを1ページに集約）

## 3. コア機能
- [ ] ファーストビュー（店名アニメーション→LP表示）
- [ ] Aboutセクション
- [ ] 人気メニュー表示（5品）
- [ ] 店内紹介
- [ ] 基本情報（営業時間・定休日・席数・TEL）
- [ ] アクセス
- [ ] 予約フォーム
- [ ] 予約CTA（電話）
- [ ] フッター（SNSリンク含む）
- [ ] スクロールアニメーション
- [ ] 装飾要素（あしらい）

## 4. データモデル設計
データベース不要（静的LPのため）

## 5. バックエンド/外部連携計画
- 予約フォーム: Readdyフォーム機能を使用
- SNS: Instagram・ホットペッパービューティー（外部リンク）
- Google Maps: 埋め込みiframe

## 6. 開発フェーズ計画

### フェーズ1: LP全体の構築
- 目標: 全セクションを含む1ページLPの完成
- 成果物: デザイン・アニメーション・フォーム・全セクションが実装されたLP
